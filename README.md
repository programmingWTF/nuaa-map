# NUAAMap — 南京航空航天大学天目湖校区智能地图

为南航天目湖校区新生及访客提供交互式校园地图与智能问答服务。

## 核心功能

- **校园地图展示**：基于矢量/栅格地图呈现天目湖校区全貌
- **建筑详情查询**：点击建筑查看名称、功能、开放时间等详细信息
- **智能问答助手**：新生可通过自然语言提问，获取校园相关信息

## 技术栈（建议）

- **前端**：React / Vue + Leaflet.js / MapLibre GL JS
- **后端**：Python FastAPI / Node.js Express
- **AI/智能体**：LangChain + RAG（检索增强生成）
- **数据**：GeoJSON 格式存储地理信息
