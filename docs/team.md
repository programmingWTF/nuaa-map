# 团队通讯录与职责

## 各组职责详情

### ① 地图图纸与美工 — 刘玉涵、陈顾云
- **手绘**天目湖校区平面地图（鸟瞰图风格）
- 将手绘地图高清扫描或拍摄，交付电子版（≥ 4K 分辨率）
- 设计网站 UI（配色、布局、图标）
- 设计建筑详情面板样式
- 设计 AI 聊天界面样式
- 输出：手绘地图扫描图（`assets/map/`）、UI 设计稿、图标素材

### ② 交互功能 — 刘楚江、薄天乙
- 以手绘地图扫描图为底图，叠加建筑点击热区
- 地图缩放、拖拽等交互逻辑
- 建筑详情弹窗/Panel 组件
- AI 聊天界面的前端交互
- 前端状态管理
- 需要对接：③组的 API、⑥组的像素坐标数据（`data/positions/`）

### ③ 平台搭建 — 罗羽彤、李桂聿、郑应楠
- 前端项目脚手架搭建
- 后端 API 服务器搭建
- 数据库设计与维护
- CI/CD 流程配置
- 部署与运维
- 需要对接：所有组的集成工作

### ④ 调研与数据收集 — 李睿、蒋荀莉、张聿修、王小雯
- 校园建筑信息实地调研
- 建筑功能、开放时间、设施信息采集
- FAQ 问题与答案整理
- 现场拍摄建筑照片
- 输出：建筑信息 JSON 文件（`data/raw/`，纯文字信息，不含坐标）

### ⑤ 智能体训练 — 胡德文、李睿、姜禹锡
- AI 问答模型选型与部署
- RAG 检索增强生成管道搭建
- 知识库构建与维护
- Prompt 工程优化
- 智能体 API 接口开发
- 需要对接：④组的数据（文字信息）、③组的部署平台

### ⑥ 数据转换 — 杜明泽
- 在①组的手绘地图扫描图上，标注各建筑的**像素坐标**
- 标注建筑的可点击区域（矩形/多边形）
- 将④组的文字数据与像素坐标合并为完整数据
- 可自制简易坐标拾取工具辅助标注
- 数据质量检查
- 输出：`data/positions/`（含完整信息 + 像素坐标的 JSON）
- 需要对接：④组（输入文字数据）、①组（输入手绘地图）、②组（输出带坐标数据）
